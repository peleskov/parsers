Salewa NXT 32 - прочный универсальный рюкзак, идеально подходящий как для туризма, так и для разнообразных альпинистских вылазок в самых экстремальных горных условиях. Сочетая в себе инновационные технологии, функциональность и долговечность, этот рюкзак станет вашим незаменимым спутником в любых приключениях в горах.

Рюкзак изготовлен из двух сверхпрочных материалов: ультралёгкой ткани Challenge ULTRA™ 100X, обеспечивающей исключительную устойчивость к истиранию и разрывам, с водостойкой подкладкой из переработанного полиэстера, а также высокопрочного, проверенного в экспедициях нейлона Robic® плотностью 330D.

Облегающая система переноски Dry Back Contact, совместимая с обвязкой и оснащённая раздельными плечевыми лямками Split Shoulder Straps, обеспечивает стабильность, свободу движений и отличную вентиляцию. Рюкзак оснащён полным набором функций для технического альпинизма, включая крепления для двух ледорубов, держатель для шлема, совместимость со стропами для фиксации лыж, двусторонний боковой доступ к основному отделению, боковой карман на молнии и фиксацию верёвки.

  • Совместимость со стропами для диагональной фиксации лыж (продаётся отдельно).
  • Совместимость с сеткой-держателем для шлема (продаётся отдельно).
  • Съёмный поясной ремень.
  • Боковой доступ к основному отделению с двойными бегунками.
  • Съёмный эластичный карман на плечевом ремне.
  • Раздвоенные плечевые лямки для улучшенной эргономики и вентиляции.
  • Точки крепления для различного снаряжения.
  • Боковые направляющие для питьевой системы.
  • Сигнальный свисток для экстренных ситуаций.
  • Боковые компрессионные ремни.
  • Водоотталкивающие молнии.
  • Алюминиевые пряжки для максимальной износостойкости.
  • Крепления для двух ледорубов.
  • Пряжки, удобные для использования в перчатках.
